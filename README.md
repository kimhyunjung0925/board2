
# 🖥 springboot-project
기본 CRUD, 게시판 기능, 회원가입 기능 구현

## 📆 개발기간
2024.02.13 ~ 

## ⚙ 개발환경
#### ● IntelliJ 
#### ● JAVA 11
#### ● SpringBoot 
#### ● thymeleaf, javascript, html5, css 
#### ● Mysql, JPA Repository

## 주요기능 소개

#### CRUD
- JPA Repository를 사용한 기본 CRUD 기능 구현
- @PageableDefault를 사용한 간단한 페이징 처리
